X Marks The Plot

Welcome to our Garden Map Project Website. We created this website to have a central place to store information on community gardens in the East Bay of California. Through this website you can learn more about community gardens near you, plots of land to use for a potential community garden, resources on how to build a garden, and more information on food policy.
